  Core Payment Processing

  - Accept payments from customers across multiple blockchains (EVM, Sui, Solana, Bitcoin)
  - Process cryptocurrency transactions and verify on-chain claims
  - Generate payment links for products and services
  - Multi-token payments (accept various cryptocurrencies)

  Merchant Dashboard Features

  Wallet Management
  - View merchant wallets across different blockchains
  - Import/generate new wallets
  - Display balance with USD conversion
  - QR codes for sharing addresses
  - Sweep functionality for derived wallets

  Token Management
  - List deployed tokens (ERC-20, SPL, Move contracts)
  - Display token metadata (name, symbol, supply, decimals)
  - View token holders
  - Burn/mint operations
  - Token contract verification

  Smart Contracts
  - Connect and manage deployed smart contracts
  - ABI explorer for smart contract functions
  - Call contract functions directly
  - Contract interaction interface

  Product Management
  - Create product catalog
  - Set pricing for products
  - Link products to payment processing
  - Inventory tracking
  - Sales analytics

  Transaction History
  - Complete payment history tracking
  - Filter by status (pending, confirmed, failed)
  - Filter by blockchain
  - Search by transaction hash
  - Export to CSV/JSON

  Account Features
  - Merchant account settings
  - API key management
  - Webhook configuration
  - Team role management (owner, admin, user)
