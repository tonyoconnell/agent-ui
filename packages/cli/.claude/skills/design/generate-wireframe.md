# Generate Wireframe

**Category:** design
**Version:** 1.0.0
**Used By:** agent-designer

## Purpose

Creates wireframe from feature spec.

## Output

```markdown
# Landing Page Wireframe

## Layout
- Header (logo, nav)
- Hero (heading, CTA)
- Features (3-col grid)
- Pricing (table)
- Footer

## Components
- Button (primary, secondary)
- Card (with image, title, description)
```
